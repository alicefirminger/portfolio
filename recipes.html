Project: random recipe generator

Tech stack used:

Brief:

This project is a hackathon from week 6 of School of Code. In a team of 3, we used a recipe API to create a random recipe generator in one day.

App features:

  1. Click a button to reveal a random recipe
  2. An recipe image and method displays on the screen
  3. Click a button to close the recipe
  4. Generate as many random recipes as you want

What I learned:

  1. Using useEffect to fetch from APIs
  2. Getting data from objects using dot notation
