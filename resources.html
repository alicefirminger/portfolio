Brief:

In a team of 4, we came up with a problem that School of Code students may have in common and created an app based solution in one week.

Problem Statement: Students' want a single source where they can easily access previous learning resources and tech links

Proposed solution: A resource website where students can find previous resources or links and filter by topic.

App features:

  1. List of tech resources displayed on landing page
  2. Clickable image links
  3. Dropdown menu to filter resources by topic
  4. Results bar that lets you know how many results are found in your search
  5. In the future there will be an option to submit your own resource to the site

What I learned:

  1. The importance of keeping our MVP simple.
  2. Using git branching and resolving merge conflicts.
  3. As a team, we learned how to use MaterialUI components.
  4. Time management and delagation strategies, using trello for tickets.
