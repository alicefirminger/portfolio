Project: flag finder

Tech stack used:

Brief:

In a team of 3 we were asked to recreate a flag finder app from an example site in one day. We used a map API to fetch the data to be used.

App features:

  1. View all the worlds flags on the page
  2. Filter by region e.g. Europe
  3. See information about each country under their flags
  4. View the site on different devices

What I learned:

  1. Intialising react projects and the use of useStates/props
  2. Fetching data from APIs
  3. Recreating the functionality and styling of another site
  4. Using CSS flexbox to make more responsive projects
